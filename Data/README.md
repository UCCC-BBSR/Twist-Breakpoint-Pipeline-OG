# hayashi_sarcoma_fusion_files

## Project Notes

Part of the Project Collection `Identification of gene fusions`  

Location of shared sarcoma fusion files used in the later fusion analyses. Prior to this directory, some of these files were included within each analysis directory.  


## Contents

| File | Description |
|---------|---------|
| `igv_session_template.xml` | IGV template used to fill in breakpoint information  |
| `primer3_template` |  Primer3 template to fill in breakpoint information |
| `temp_primer3` |  Primer3 template to fill in breakpoint information |
| `Probes_merged_ok_CU-Hayashi_PedSolidTumorTransloc_TE-92680772_hg38_250404204242.bed` | Bed file of Twist assay pulldown regions  |
| `targets_CU-Hayashi_PedSolidTumorTransloc_TE-92680772_hg38.bed` | Bed file of DESIRED twist assay regions (what we wanted to pull down, not necessarily what Twist was designed)  |
| `primer3_template` |  Primer3 template to fill in breakpoint information |
| `summary_table_column_descriptions_twist.xlsx` | Description of columns in summary tables (adopted for Twist-enrichment)  |
| `summary_table_column_descriptions.xlsx` |  Description of columns in summary tables |
| `WholeGenomeFasta` |  Folder of hg38 genome files |
